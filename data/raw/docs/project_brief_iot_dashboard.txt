Project Brief: IoT Sensor Monitoring Dashboard

Project Overview:
A real-time monitoring platform for industrial IoT sensors with alerting and analytics.

Actors:
- Plant Engineer, Operations Manager, Maintenance Technician, System Admin

Functional Requirements:
1. Sensor data ingestion via MQTT/REST supporting 1000+ devices simultaneously.
2. Real-time dashboard with live charts, threshold indicators, and device status.
3. Alert management: configurable thresholds, escalation rules, multi-channel notifications.
4. Historical data viewer with time-range selection, aggregation, and data export.
5. Device management: registration, firmware version tracking, health monitoring.
6. Predictive maintenance module using anomaly detection on sensor trends.

Non-Functional Requirements:
7. Ingest and process at least 10,000 sensor readings per second without data loss.
8. Dashboard must refresh within 2 seconds of new data arrival.
9. Data retention: raw data for 90 days, aggregated data for 3 years.
10. End-to-end encryption for all device communication (TLS 1.3 minimum).
