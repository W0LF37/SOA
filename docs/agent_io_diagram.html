RiskAnalyzer
src/agents/risk_analyzer.py · analyze()
🔵 LLM Optional
Inputs
  • TaskList
  • plan_summary: dict
  • CriticReport optional
6 Rule-based categories:
1. Bottleneck — 5+ downstream tasks
2. Complexity — C5 on critical path
3. Schedule — >300h total / 7+ parallel
4. Dependencies — inferred non-explicit
5. Resources — >60% effort one role
6. Quality — critic score < 0.80

+ LLM novelty layer:
2–3 novel risks not captured by rules
Tagged source="llm"
Outputs
  • RiskReport
risk_level: low | medium | high | critical
risk_score: 0.0 – 1.0
total_risks: int
risks[]: RiskItem
  ├─ category: str
  ├─ severity: critical|high|medium|low
  ├─ message: str
  ├─ affected_tasks: list[str]
  ├─ mitigation: str
  └─ source: "rule" | "llm"
mitigations[]: list[str]
Risk score penalties:
critical +0.30 / high +0.15
medium +0.05 / low +0.01

Level thresholds:
≥ 0.70 → critical  ·  ≥ 0.40 → high
≥ 0.20 → medium  ·  else → low

MonitorAgent
src/agents/monitor.py · track_progress()
⚪ No LLM
Inputs
  • TaskList
  • repo_path: str optional
  • commits: list[dict] optional
Matching — Primary:
SentenceTransformer all-MiniLM-L6-v2
Cosine similarity threshold: 0.65
Skip commit if title < 10 chars

Matching — Fallback:
Keyword match (stop words filtered)
Against commit msg + files_changed

Status inference:
0 commits → not_started (0.0)
1 commit + stale → in_progress (0.3)
1 commit + recent → in_progress (0.5)
1 commit + done-kw → completed (1.0)
2 commits, no done-kw → in_progress (0.7)
3+ commits → completed (1.0)
Outputs
